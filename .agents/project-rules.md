They also offer general tech support services, coming into your home and teaching you how to use things, troublehsoot and fix devices, or other general tech support services. Where we charge a per device fee to clean up unused files and programs, disable services that are not being used, and optimize the device for better performance. Even fixing a corrupted os or crashing pc. those are 3 distinct services.

Technical overview:
The site is a next.js site. The routes should be a home page, a page for each service, a pricing page, a dynamic town based page for seo.
The tech stack is lite, using as little dependancies as possible, only for core functionality like calendar, emails, and payments will we add packages to the package.json.
The site is using Google Calender and Google Meets API for online meeting. Site will use stripe to accept payments for $10 for appointment confirmations.
